# Інструкція з налаштування (для розробників)

Цей документ пояснює, як розгорнути проєкт "Studium" локально для тестування або розробки.

## Необхідні інструменти

* **Веб-браузер** (Chrome, Firefox тощо).
* **Python** (версії 3.10+).
* **Git** (для клонування репозиторію).
* **Обліковий запис** на платформі (для тестування функцій).

*(Це приклад для проєкту, який запускається локально)*

## Кроки встановлення (Макет)

1.  **Клонуйте репозиторій:**
    ```bash
    git clone [https://github.com/Andrii/documentation-lab-Andrii.git](https://github.com/Andrii/documentation-lab-Andrii.git)
    ```
2.  **Перейдіть у папку проєкту:**
    ```bash
    cd documentation-lab-Andrii
    ```
3.  **Встановіть залежності (приклад для Python):**
    ```bash
    pip install -r requirements.txt
    ```
4.  **Запустіть сервер:**
    ```bash
    python manage.py runserver
    ```

## Перевірка роботи

Якщо ви все зробили правильно, ви зможете отримати доступ до платформи у вашому браузері.

1.  Відкрийте браузер.
2.  Перейдіть за адресою: `http://127.0.0.1:8000/`.
3.  Ви повинні побачити головну сторінку або сторінку входу на платформу "Studium".
