# Огляд API "Studium" (v1)

Цей документ описує базові API endpoints для взаємодії з платформою. Усі відповіді повертаються у форматі JSON.

## 1. Отримати список усіх курсів

* **Endpoint:** `GET /api/courses`
* **Опис:** Повертає список усіх доступних курсів на платформі.
* **Успішна відповідь (Код 200):**
    ```json
    [
      {
        "id": 1,
        "title": "Вступ до Python",
        "instructor": "Професор Коваленко"
      },
      {
        "id": 2,
        "title": "Історія України",
        "instructor": "Професор Петренко"
      }
    ]
    ```

## 2. Записати студента на курс

* **Endpoint:** `POST /api/courses/{id}/enroll`
* **Опис:** Записує поточного (авторизованого) студента на курс із вказаним `{id}`.
* **Тіло запиту:** (Порожнє)
* **Успішна відповідь (Код 200):**
    ```json
    {
      "status": "success",
      "message": "Ви успішно записані на курс 'Вступ до Python'"
    }
    ```
* **Помилка (Код 404):**
    ```json
    {
      "status": "error",
      "message": "Курс не знайдено"
    }
    ```

## 3. Коди помилок

* **200 (OK):** Запит успішний.
* **401 (Unauthorized):** Помилка автентифікації (користувач не увійшов у систему).
* **404 (Not Found):** Ресурс (курс або endpoint) не знайдено.
